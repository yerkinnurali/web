# Қазақ Тағамдары - Тапсырыс беру жүйесі

Қазақ тағамдарын тапсырыс беруге арналған веб-қолданба.

## Ерекшеліктер

- **Қазақ тағамдары**: Бешбармак, қазы, бауырсақ, манты, плов және басқалары
- **Қазақ дизайны**: Қазақстан туының түстері (көк және алтын)
- **Тапсырыс беру**: Онлайн тапсырыс беру жүйесі
- **Фильтрлеу**: Тағамдарды категория бойынша сұрыптау
- **Қазақ тілі**: Барлық интерфейс қазақ тілінде

## Орнату

1. **Базаны құру**:
   - phpMyAdmin-ды ашыңыз: `http://localhost/phpmyadmin`
   - `data/schema.sql` файлын импорттаңыз
   - Немесе `quick_add_items.sql` файлын орындаңыз

2. **Баптаулар**:
   - `includes/config.php` файлында база деректерін тексеріңіз

3. **Қолдану**:
   - `http://localhost/index.php` - Басты бет
   - `http://localhost/portfolio.php` - Тағамдар мен тапсырыс беру

## Тағамдар

### Қолжетімді тағамдар:
- **Бешбармак** - Дәстүрлі қазақ тағамы
- **Қазы** - Дәстүрлі қазақ қазысы
- **Бауырсақ** - Алтын түсті бауырсақтар
- **Манты** - Буға пісірілген манты
- **Плов** - Дәстүрлі қазақ пловы
- **Шашлық** - Дәмді қазақ шашлығы
- **Құрт** - Дәстүрлі қазақ құрты
- **Қауын** - Жазғы қазақ қауыны

## Тапсырыс беру

1. `portfolio.php` бетіне өтіңіз
2. "Тапсырыс беру" бөлімінде форманы толтырыңыз
3. Тағам түрін таңдаңыз
4. Күн мен уақытты көрсетіңіз
5. "Тапсырыс Жіберу" батырмасын басыңыз

## Дизайн

Сайт қазақстандық дизайнмен жасалған:
- **Көк түс** (#0066cc) - Қазақстан туының көк түсі
- **Алтын түс** (#ffd700) - Қазақстан туының алтын түсі
- **Қараңғы тақырып** - Қазіргі заманғы қараңғы дизайн
